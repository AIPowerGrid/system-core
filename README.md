# AI Power Grid

A massive crowdsourced distributed inference network for AI Models. This platform supports both Image and Text generation, enabling users without a powerful GPU to leverage technologies like Stable Diffusion or Large Language Models by utilizing spare/idle resources from the community. It also facilitates non-python clients, such as games and apps, to harness AI-driven generations.

For more questions, check the [FAQ](FAQ.md)

# Registering

To participate in the grid, you need to have a registered account, or use anonymous mode.

To register an account, visit the AI Power Grid website:
   * [AI Power Grid Registration](https://api.aipowergrid.io/register)

and sign in with one of the available services. After signing in, you'll be prompted to enter a username. Once submitted, it will automatically create a user object for you and provide an API key for identification.

Keep this API key safe and use it for your client or bridge.

By signing in first, you have the flexibility to change your username and API key at any time.
We do not store any identifiable information other than the ID string sent by the oauth for your user. This is solely used for ensuring user uniqueness.

If you prefer, you can also create a pseudonymous account without using oauth. However, *we will not maintain such accounts*. Losing access means creating a new one. If someone else acquires your API Key, they can impersonate you. Changes to the username or API key will not be possible. If these risks are unacceptable, please sign in using one of the available services.

For those who wish to use the service anonymously without a pseudonymous account, '0000000000' can be used as your API key. Note that your usage and contributions will not be tracked. Be aware that if the service becomes too overloaded, anonymous mode may be disabled!

The purpose of registering is to monitor your usage and contributions. The more you contribute to the Grid, the higher your priority.
